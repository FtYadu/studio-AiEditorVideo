# **App Name**: AIVidFlow

## Core Features:

- Workflow Editor: Intuitive node-based workflow editor for video processing.
- Timeline View: Real-time timeline view for precise video editing and arrangement.
- Auto Scene Detection: AI-powered scene detection tool to automatically identify and segment video scenes.
- Auto Captioning: Automatic transcription tool that converts video audio into text, with options for styling and export.
- Smart Templates: AI-powered tool to generate punch-cut edits based on templates, using provided templates as a tool for guiding edits.
- Asset Management: Asset management system with smart bins.
- Performance Optimization: Support for draft and full quality previews, optimized through proxy management.

## Style Guidelines:

- Primary color: Deep Indigo (#6366F1) to convey intelligence and sophistication in the video editing process.
- Background color: Dark Navy (#0F172A) to reduce eye strain during long editing sessions.
- Accent color: Electric Purple (#A855F7) for interactive elements and highlights.
- Headline font: 'Space Grotesk' sans-serif, for a modern, computerized style, for all headers and titles.
- Body font: 'Inter' sans-serif, for a clean, readable style.
- Use consistent, modern icons from Lucide to maintain a clean and professional look.
- Maintain a modular layout with clear divisions for tool panels and workspace.
